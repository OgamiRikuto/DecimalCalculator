# 10進計算機

c言語で作成された10進計算機です。  
10進数を用いて計算するため、計算誤差がでません。  
  
四則演算のみ可能です。  
()は使用できます。

## 実行方法(macOS)

Makefileのあるディレクトリにて

```bash
$ make
```

とするとコンパイルされます。

```bash
$ make test
```

とすると実行できます。   

コンパイル時に作成されるオブジェクトファイルと実行ファイルを削除したい場合は、
```bash
$ make clean
```
を実行して下さい。  

## 終了について
強制終了してください.  
ctrl+cでできます.

## 入力について
プログラムは"="が入力されるまで入力を受け付けます。  
"="が入力され、<u>enterが押される</u>と、入力の受付を終了し、計算結果を出力します。　
  
計算結果が出力された後はまた入力を受け付けます。
終了はCtrl+Dを押してください。

**式に構文エラーがある場合**  
エラー文を出力し、プログラムを強制終了します。

エラー文一覧
- `Error: Invalied character:`      : 式に使用できない文字が使用されています。
- `Error: No tokens found.`         : 式の形式に不備があります。
- `Error: mismatched parentheses`   : ()の数が正しくありません。
- `Error: insufficient operands`    : 演算子が正しくありません。
- `Error: divide by zero`           : 0除算をしようとしています。
- `Error: invalid RPN expression`   : 式が不正です。

## 出力について

除算が式に含まれている場合、デフォルトでは小数点以下2桁まで表示するようになっています。  
小数点以下の桁数を変更したい場合は、

```bash
$ make n=10 test
```
のように`n`に指定したい桁数を代入してください。

また、このプログラムは0のトリミングを行っていません。
